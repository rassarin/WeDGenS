=======================================================================
 Gruntによるunderscoreテンプレート生成方法
=======================================================================

・事前準備
  - node.js(http://nodejs.org/)をインストールする。

  - 下記コマンドを実行し、Gruntをインストールする。

    npm install -g grunt-cl

  - scripts/build/backbonejsディレクトリで下記コマンドを実行し、
    プラグインをインストールする。

    npm install

・scripts/build/backbonejsディレクトリで下記コマンドを実行する。

  grunt build

  --> public/js/templates/jst.jsが生成される。
